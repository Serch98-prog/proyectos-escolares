# Proyectos Escolares

Repositorio donde almaceno los programas desarrollados durante mi formación en Ingeniería en Desarrollo de Software en Tecmilenio.

## Objetivo

Este repositorio contiene proyectos desarrollados durante mi formación en Ingeniería en Desarrollo de Software, con el propósito de practicar lógica de programación y Python.

## Habilidades desarrolladas

- Variables
- Condicionales (if, elif, else)
- Entrada y salida de datos
- Operadores matemáticos
- Validación básica de datos

## Proyectos

### Calculadora básica
Realiza operaciones de suma, resta, multiplicación y división.

### Sistema de membresía
Determina descuentos según el tipo de membresía del cliente.

### Sistema de ventas de videojuegos
Calcula compras, descuentos y totales de venta.

### Taquilla simple
Gestiona la venta de boletos aplicando promociones según la cantidad adquirida.
